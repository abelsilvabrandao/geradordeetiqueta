Gerador de Etiquetas

Como usar o Gerador de Etiquetas

  1. Prepare seu arquivo Excel (.xlsx) seguindo o formato padrão abaixo
  2. Clique em "Escolher arquivo" e selecione seu arquivo Excel
  3. Clique em "Visualizar" para conferir as etiquetas geradas
  4. Verifique se todos os dados estão corretos na visualização
  5. Clique em "Gerar PDF" para baixar o arquivo para impressão

Formato do Arquivo Excel

Na primeira linha do Excel, configure o cabeçalho exatamente assim:

  • Coluna A: Navio__VG_ (Nome do navio)
  • Coluna B: ETA (Data no formato DD/MM/AAAA)
  • Coluna C: CLIENTE (Cliente/Importador)

Atenção:

  • A partir da linha 2, insira os dados em CAIXA ALTA (MAIÚSCULO)
  • Evite espaços em branco antes e depois dos textos
  • O nome das colunas deve ser EXATAMENTE como mostrado acima
  • Datas devem estar no formato DD/MM/AAAA (exemplo: 09/06/2025)
